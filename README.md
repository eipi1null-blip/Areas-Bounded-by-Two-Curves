# Areas Bounded by Two Curves

Interactive applet illustrating area between two curves using

`integral f(x) dx - integral g(x) dx`.

## GitHub Pages Deployment

These files must be placed directly in the repository root:

- `index.html`
- `area-between-curves-applet.html`
- `.nojekyll`
- `.github/workflows/deploy-pages.yml`
- `README.md`

In GitHub, use:

1. `Settings > Pages`
2. `Source: GitHub Actions`
3. Push these files to the `main` branch
4. Wait for the Pages workflow to complete under `Actions`

The deployed page should be:

`https://eipi1null-blip.github.io/Areas-Bounded-by-Two-Curves/`
